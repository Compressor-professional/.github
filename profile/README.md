# Compressor Software — Batch Video Encoding with Professional Settings for Video Engineers

## About Compressor Software

Compressor Software provides a complete batch video encoding environment with professional settings for video engineers, broadcast technicians, and post-production facilities. Compressor Software transcodes ProRes, HEVC, H.264, MXF, MPEG-2, and HLS streams with GPU acceleration. Compressor Software offers distributed encoding across multiple computers, custom presets, and watch folder automation. Compressor Software enables users to create delivery packages for Apple devices, streaming platforms, broadcast, web, and archive. Compressor Software includes closed caption embedding, job chaining, watermarks, filters, and chapter markers. Compressor Software supports batch renaming, image sequence output, and Dolby Vision metadata injection. Compressor Software is trusted by video engineers, post-production houses, broadcast engineers, and media asset managers worldwide.
